Neuropass with MATLAB

1. Collect a data sequence using the Emotiv EEG headset and save the collected data as a CSV file.

2. Run main.m by navigating to this folder in MATLAB and typing in the command 'main'.

3a. If creating a password for the first time, type 'a' at the prompt. 
3b. If an existing password has already been set, type 'b' at the prompt.

4. Navigate through the file browser and select the new CSV file.

5a. Neuropass will either save the new password.
5b. Neuropass will confirm the password or reject it.
